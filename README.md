# Wireframe Interpretation - Responsive Landing Page

## Project Overview

This project is a responsive landing page created for a midterm exam assignment. The objective was to build a responsive landing page based on a provided wireframe and submit it along with screenshots for evaluation.

**Project Name:** NEXOVATE Landing Page  
**Type:** Responsive Website / Landing Page  
**Technology:** HTML, CSS, JavaScript  
**Date:** April 2026

---

## Features

- **Responsive Design** - Adapts to different screen sizes (mobile, tablet, desktop)
- **Navigation** - Responsive navigation with hamburger menu for mobile
- **Hero Section** - Main landing area with headline, description, and call-to-action button
- **Geometric Visual Elements** - Decorative triangular shapes in the design
- **Clean & Modern UI** - Professional appearance with minimalist aesthetics
